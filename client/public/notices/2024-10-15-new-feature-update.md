# 새로운 기능 업데이트

토론 플랫폼에 새로운 기능들이 추가되었습니다!

## 추가된 기능

### 1. 토론 요약 기능
- AI가 토론 내용을 자동으로 요약해드립니다
- 온라인 토론에만 제공됩니다
- 토론 생성자만 요약을 생성할 수 있습니다

### 2. 클립보드 복사 기능
- 토론 요약을 클립보드에 복사할 수 있습니다
- 마크다운 형식으로 복사됩니다

### 3. 시간대 개선
- 모든 시간이 한국 시간대로 정확히 표시됩니다

## 사용 방법
각 기능에 대한 자세한 사용법은 도움말을 참고해주세요.
